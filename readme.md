# Braindump

## Descrizione

Braindump è un progetto che mira a facilitare la raccolta e l'organizzazione delle idee, delle esperienze e delle conoscenze in un formato strutturato. Utilizzando un'interfaccia CLI migliorata, il progetto consente agli utenti di interagire con i dati in modo intuitivo e efficiente.

## Funzionalità

- **Interfaccia CLI**: Un'interfaccia a riga di comando user-friendly per l'interazione con il sistema.
- **Modalità Interattiva**: Supporto per modalità interattive che migliorano l'esperienza utente.
- **Gestione degli Errori**: Meccanismi di recupero da errori per garantire un'esperienza fluida.
- **Analisi dei Tempi di Esecuzione**: Monitoraggio delle prestazioni dei test con visualizzazione tabellare.
- **Documentazione Completa**: Documentazione dettagliata del codice e delle funzionalità.

## Installazione

1. Clona il repository:
   ```bash
   git clone https://github.com/DSmanuelvimercati/braindump.git
   ```
2. Naviga nella directory del progetto:
   ```bash
   cd braindump
   ```
3. Installa le dipendenze:
   ```bash
   pip install -r requirements.txt
   ```

## Utilizzo

Per avviare il progetto, utilizza il seguente comando:
```bash
python main.py
```

Segui le istruzioni visualizzate per interagire con il sistema.
